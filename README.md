<h1 align="center">BlenderBot AI Chat Bot</h1>

<p align="center">
A conversational AI chatbot built using Python, Hugging Face Transformers, and Gradio.
</p>

<hr>

<h2>Project Overview</h2>

<p>
This project implements an AI-powered chatbot using the 
<b>facebook/blenderbot-400M-distill</b> conversational model. 
The chatbot allows users to interact with an AI system capable of generating 
context-aware responses and maintaining conversation history.
</p>

<p>
The model is a distilled version of Facebook's BlenderBot architecture designed 
for open-domain conversations while remaining lightweight enough to run on local 
machines. It uses transformer-based neural networks trained for conversational 
tasks and dialogue generation. :contentReference[oaicite:0]{index=0}
</p>

<hr>

<h2>Features</h2>

<ul>
<li>AI conversational chatbot</li>
<li>Context-aware responses with conversation history</li>
<li>Web-based GUI using Gradio</li>
<li>Powered by Hugging Face Transformers</li>
<li>Runs locally with minimal setup</li>
<li>Open-source and customizable</li>
</ul>

<hr>

<h2>Technologies Used</h2>

<h2>Installation</h2>

<p>Clone the repository:</p>

<pre>
git clone https://github.com/faique1234/BlenderBot-AI-Chat-Bot-
cd BlenderBot-AI-Chat-Bot-
</pre>

<p>Install required dependencies:</p>

<pre>
pip install transformers torch gradio
</pre>

<hr>

<h2>Usage</h2>

<p>Run the chatbot:</p>

<pre>
python blenderbot_(ai_chat_bot).py
</pre>

<p>
After running the script, a local interface will open in your browser 
(typically at <b>http://127.0.0.1:7860</b>) where you can interact with the chatbot.
</p>
